## BOM
- 8x DSA keycaps
- 8x Cherry MX Switches
- 8x SOM123 Diodes
- 4x M3x16 Bolt
- 4x M3 Heatset
- 2x EC11 Rotary Encoder
- 1x XIAO RP2040
